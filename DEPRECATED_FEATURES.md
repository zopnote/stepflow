# List of deprecated features
The features listed here will be removed in the next major version ``3.0.0``.
> Do not worry, I'll work on 
> better documentation to support migration and new users.
> The new documentation will be available in
> the next major version as well.

**File removals:**
* ``lib/platform.dart``
* ``lib/cli.dart``
* ``lib/common.dart``
* ``lib/src/response.dart``
* ``lib/src/platform/**``
* ``lib/src/io/command.dart``
* ``lib/src/io/flag.dart``
* ``lib/src/io/steps/log_print.dart``
* ``lib/src/core/bubble_step.dart``

**Symbols:**
* ``Command``, ``Flag``, ``BoolFlag``, ``TextFlag``
* ``Bubble``
* ``FlowContextController``
* ``LogColor``, ``LogASCIIContext``
* ``Response``, ``Level``

+ _All platform tooling-related symbols._
